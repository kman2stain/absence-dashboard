📂 File Naming Rule

Every report file must include the vehicle number in brackets:

StockReport(TRK-001).xlsx
LoadingPlan(TRK-001).xlsx
BinReport(TRK-001).xlsx

The text before the bracket can be anything — only (VehicleNumber) is used to match the file.

🔄 Daily Workflow
1️⃣  Upload all 3 report files to their Drive folders
2️⃣  Activate the vehicle in the Vehicles tab above
3️⃣  Employees open the app and select the vehicle
4️⃣  Employees scan barcodes and confirm each putaway
5️⃣  Employee taps the session counter → Submit Report
6️⃣  Report saved automatically to the Output folder
7️⃣  Deactivate the vehicle when putaway is complete
📋 Required Column Names
Stock Report:
  CARTON_BARCODE, POLI, PO, PO_ITEM
Loading Plan:
  PO, LI or PO_ITEM (POLI column also works)
Bin Report:
  POLI, BIN LOCATION (or BIN), QTY (or AVAILABLEQTY)
